# 초음파 기초 원리

## 파동이란

물에 돌을 던지면 물결이 퍼져나간다. 이처럼 **어떤 진동이 주변으로 퍼져나가는 현상**을 파동(Wave)이라고 한다.  
파동은 에너지를 전달하지만, 매질(물·공기·고체) 자체가 이동하는 것은 아니다. 입자들이 제자리에서 진동하면서 그 에너지를 이웃 입자로 넘겨줄 뿐이다.

```
돌을 던진 지점
     ↓
  ≋≋≋≋≋≋≋≋≋≋≋   ← 물결(에너지)이 퍼져나감
  물 입자들은 위아래로 진동할 뿐, 옆으로 이동하지 않음
```

파동을 설명하는 핵심 개념은 세 가지다: **주파수**, **파장**, **진폭**.

---

## 초음파란 무엇인가

음파는 공기·액체·고체 등 매질 속에서 입자가 진동하며 전파되는 파동이다.  
인간의 귀가 들을 수 있는 범위는 **20 Hz ~ 20,000 Hz(가청 주파수)** 이며, 초음파는 이보다 높은 주파수 대역의 음파다.

| 구분              | 주파수 범위          | 예시                         |
| ----------------- | -------------------- | ---------------------------- |
| 가청음            | 20 Hz ~ 20 kHz       | 사람 대화, 악기              |
| 초음파            | 20 kHz 이상          | 박쥐 탐지, UT 검사           |
| UT 검사 사용 범위 | **0.5 MHz ~ 20 MHz** | 결함 크기·해상도에 따라 선택 |

> 주파수가 높을수록 파장이 짧아져 작은 결함을 탐지하기 유리하다.  
> 단, 주파수가 높을수록 **감쇠(감소)** 도 빨라져 깊은 곳까지 도달하기 어렵다.

---

## 주파수 (Hz, 헤르츠)

주파수는 **1초 동안 진동이 몇 번 반복되는가**를 나타내는 값이다.  
단위는 **Hz(헤르츠)** 이며, 독일 물리학자 하인리히 헤르츠(Heinrich Hertz)의 이름에서 왔다.

```
1초 동안의 진동 횟수 = 주파수 (Hz)

1 Hz  → 1초에 1번 진동
100 Hz → 1초에 100번 진동
1 kHz → 1초에 1,000번 진동   (k = 킬로 = 1,000)
1 MHz → 1초에 1,000,000번 진동  (M = 메가 = 100만)
```

**일상 예시:**

| 대상             | 주파수                |
| ---------------- | --------------------- |
| 저음 베이스 기타 | 약 40 Hz              |
| 사람 목소리      | 100 ~ 3,000 Hz        |
| 모기 날개 소리   | 약 600 Hz             |
| 인간 가청 한계   | 약 20,000 Hz (20 kHz) |
| 박쥐 초음파      | 20 ~ 100 kHz          |
| UT 검사          | 0.5 ~ 20 MHz          |

> 주파수가 높을수록 소리가 높고(고음), 낮을수록 낮다(저음).

---

## 파장 (Wavelength, λ)

파장은 **파동이 한 번 진동하는 동안 이동한 거리**다. 기호는 **λ(람다)**.

```
파장(λ) = 음속(V) ÷ 주파수(f)

        파장 λ
   ←──────────────→
   ╭──╮          ╭──╮
───╯  ╰──────────╯  ╰─── (파형)
       한 번의 진동이 이동한 거리
```

파장이 짧을수록(= 주파수가 높을수록) 더 작은 결함을 탐지할 수 있다.  
일반적으로 **파장의 절반보다 작은 결함은 탐지하기 어렵다.**

**탄소강 종파(5,920 m/s) 기준 파장:**

| 주파수 | 파장   | 탐지 가능 최소 결함(이론) |
| ------ | ------ | ------------------------- |
| 1 MHz  | 5.9 mm | 약 3 mm 이상              |
| 5 MHz  | 1.2 mm | 약 0.6 mm 이상            |
| 10 MHz | 0.6 mm | 약 0.3 mm 이상            |

---

## 진폭 (Amplitude)

진폭은 **파동이 얼마나 크게 진동하는가**, 즉 파의 높이다.  
초음파 검사에서는 진폭이 클수록 에코 신호가 강하다는 뜻이며, 보통 **%** 로 표시한다.

```
진폭 큼 (에코 강함):     진폭 작음 (에코 약함):
신호│                    신호│
   │  ██                     │
   │  ██                     │
   │  ██                     │  ██
   │  ██                     │  ██
   └──────→ 시간             └──────→ 시간

→ 큰 결함, 빔이 결함에 정면으로 반사   → 작은 결함, 비스듬히 반사
```

> 진폭의 크기만으로 결함 크기를 단정 지을 수는 없다.  
> 결함의 방향, 표면 상태, 깊이에 따라 에코 높이가 달라지기 때문이다.

---

## 주파수가 높을수록 작은 결함을 탐지할 수 있는 이유

### 회절(Diffraction) — 파동이 장애물을 "못 보는" 현상

파동이 장애물을 만났을 때, **장애물의 크기가 파장보다 훨씬 작으면** 파동은 장애물을 그냥 돌아서 지나간다. 이 현상을 **회절**이라고 한다.

```
결함 크기 << 파장일 때:          결함 크기 ≈ 파장일 때:

  →→→→→→→→→→→→→                →→→→→→→→→↑→→→
  →→→→→→→→→→→→→                →→→→→→→→→ ● ←반사
  →→→→→→→●→→→→→                →→→→→→→→→↓→→→
  →→→→→→→→→→→→→

  파동이 결함을 돌아서 진행         파동이 결함에 부딪혀 반사
  (에코 거의 없음 → 탐지 못함)      (에코 발생 → 탐지 가능)
```

### 물결파 비유

연못에 두 종류의 물결을 만든다고 가정해 보자.

```
파장 20cm인 물결 vs 파장 2cm인 물결
                   ↓
   [1cm짜리 작은 돌]을 물에 넣으면

  파장 20cm: 돌이 너무 작아서 파동이 돌 위로 그냥 넘어감
  파장 2cm : 돌의 크기가 파장과 비슷 → 파동이 부딪혀 퍼져나감
```

초음파도 동일하다. 결함 크기가 파장의 **절반보다 작으면** 초음파가 결함을 "느끼지 못하고" 그냥 통과하거나 회절하여 에코가 돌아오지 않는다.

### 탐지 조건 정리

결함을 탐지하려면 결함 크기가 파장의 절반보다 커야 한다.

```
탐지 가능 조건:  결함 크기  ≥  λ/2
```

이것이 의미하는 건, **"파장의 절반이 탐지 가능한 최소 결함 크기의 기준선"** 이라는 뜻이다.  
예를 들어 파장이 2mm라면, 1mm보다 큰 결함은 탐지할 수 있다.

여기서 파장 λ는 주파수 f에 의해 결정된다.

```
λ = V ÷ f

   음속 V는 재료가 정해지면 고정값 (탄소강 ≈ 5,920 m/s)
   주파수 f는 탐촉자를 선택할 때 결정
```

따라서 f를 높이면 λ가 작아지고, λ가 작아지면 탐지 기준선(λ/2)도 낮아진다.

**구체적인 예시 (탄소강 기준):**

| 목표            | 필요 파장(λ) | 필요 주파수 |
| --------------- | ------------ | ----------- |
| 3mm 결함 탐지   | λ < 6mm      | f > 1 MHz   |
| 1mm 결함 탐지   | λ < 2mm      | f > 3 MHz   |
| 0.5mm 결함 탐지 | λ < 1mm      | f > 6 MHz   |

→ 탐지하고 싶은 결함이 작을수록, 더 높은 주파수의 탐촉자가 필요하다.

### 주파수를 무한정 높이면 되지 않나?

주파수가 높아질수록 초음파가 재료를 통과하면서 에너지를 더 빠르게 잃는다(**감쇠, Attenuation**).  
파장이 짧아질수록 재료 내 결정립(grain) 등 미세 구조에 의한 산란이 증가하기 때문이다.

```
얕은 결함 탐지 → 고주파 (5~10 MHz) 유리: 해상도↑, 투과 깊이↓
깊은 결함 탐지 → 저주파 (1~2 MHz) 필요: 해상도↓, 투과 깊이↑
```

따라서 UT 검사에서는 **검사 대상의 두께와 요구 해상도**에 따라 주파수를 선택한다.

## 초음파로 결함을 찾는 원리

박쥐가 어둠 속에서 소리를 쏘고 반사파가 돌아오는 시간으로 장애물 거리를 파악하는 것과 같은 원리다.

### 기본 동작 방식 — 펄스-에코(Pulse-Echo)

초음파 탐촉자는 **발신과 수신을 동시에** 한다.

```
  [탐촉자]
     │  ← 초음파 발사
     ↓
┌────────────────┐  ← 재료 표면
│                │
│    ●           │  ← 결함 (내부 불연속부)
│                │
│                │
└────────────────┘  ← 바닥면(뒷면에코)
```

1. 탐촉자가 초음파 펄스를 재료 안으로 쏜다
2. 결함을 만나면 일부 에너지가 **반사(에코)** 되어 돌아온다
3. 바닥면에서도 반사된다
4. 탐촉자가 돌아온 에코를 받아 **A-scan** 으로 표시한다

### A-scan — 에코를 시간 순서로 나타낸 그래프

```
진폭(%)
 100 │
     │      ██           ← 결함 에코 (중간에 반사)
  50 │      ██      ██   ← 바닥면 에코
     │      ██      ██
   0 └──────────────────→ 시간 (= 깊이)
     발신  결함위치  바닥면
```

- **x축 = 시간**: 에코가 돌아오는 데 걸린 시간 → 깊이를 알 수 있다
- **y축 = 진폭**: 에코 세기 → 결함이 얼마나 강하게 반사했는지

### 파장과 반사의 관계

결함이 에코를 얼마나 강하게 돌려보내는지는 **결함 크기 대 파장의 비율**에 달려 있다.

```
경우 A: 결함 크기 >> λ (파장보다 훨씬 큰 결함)

  →→→→→→→→→→→→→
  →→→→→→→[  결함  ]←←←←  강하게 반사
  →→→→→→→→→→→→→

  A-scan: 진폭 높은 에코 피크 → 탐지 쉬움


경우 B: 결함 크기 ≈ λ/2 (탐지 경계선)

  →→→→→→→[결함]←←  약하게 반사

  A-scan: 진폭 낮은 에코 피크 → 겨우 탐지 가능


경우 C: 결함 크기 << λ (파장보다 훨씬 작은 결함)

  →→→→→→→·→→→→→  파동이 결함을 돌아서 진행 (회절)

  A-scan: 에코 없음 → 결함이 있어도 모름
```

### 에코가 A-scan 피크로 나타나는 원리

에코가 탐촉자에 도달하면 **마이크가 소리를 전기신호로 바꾸듯**, 탐촉자 내부의 압전소자가 음압을 전기신호로 변환한다.

```
강한 에코 → 압전소자에 강한 압력 → 큰 전기신호 → A-scan 피크 높음
약한 에코 → 압전소자에 약한 압력 → 작은 전기신호 → A-scan 피크 낮음
```

에코가 강해지는 조건은 두 가지다.

**① 임피던스 차이가 클수록**

경계면에서 두 재료의 임피던스(Z = 밀도 × 음속) 차이가 클수록 더 많이 반사된다.

```
탄소강(Z = 45,000,000) / 공기(Z = 400) → 차이 극단적 → 반사율 ≈ 100%
탄소강(Z = 45,000,000) / 물(Z = 1,500,000) → 반사율 ≈ 88%
탄소강 / 탄소강 → 차이 없음 → 반사율 = 0%
```

바닥면(재료/공기 경계)에서 에코가 크게 나타나는 이유가 이것이다. 결함도 내부가 공기나 이물질로 채워져 있어 임피던스 차이가 생기므로 반사가 발생한다.

**② 빔이 경계면에 수직으로 입사할수록**

```
수직 입사 → 반사파가 그대로 탐촉자 방향으로 돌아옴 → 에코 큼
비스듬히 입사 → 반사파가 다른 방향으로 튀어나감 → 에코 작음
```

## 게이트(Gate)

### 왜 게이트가 필요한가

A-scan에는 결함 에코, 바닥면 에코, 잡음(노이즈)이 모두 섞여 있다.  
**관심 있는 깊이 구간**의 에코만 골라서 분석하기 위한 구간 설정이 **게이트(Gate)** 다.

### 게이트의 3가지 구성 요소

```
진폭(%)
 100 │
     │           ██
  50 │─────┬─────██────┬─────  ← 임계값 (Threshold)
     │     │     ██   │
   0 └─────┴──────────┴──────→ 시간(깊이)
           │←── 폭 ──→│
         시작점      끝점
```

| 요소                   | 의미                                        |
| ---------------------- | ------------------------------------------- |
| **시작점 (Start)**     | 게이트가 감지를 시작하는 시간 위치 (= 깊이) |
| **폭 (Width)**         | 게이트가 커버하는 시간 구간                 |
| **임계값 (Threshold)** | 이 높이를 넘는 에코만 유효한 신호로 인정    |

에코가 게이트 구간 안에 들어오고 임계값을 넘어야 **"결함 감지"** 로 처리한다.

---

### 결함 탐지용 게이트

결함은 바닥면보다 얕은 곳에 있으므로 A-scan에서 **바닥 에코보다 앞에** 에코가 나타난다.

```
진폭│  ┌발신┐  ┌결함에코┐        ┌바닥에코┐
   │  └───┘  └───────┘        └───────┘──→ 시간
   │          ↑
   │    바닥 에코보다 앞에 나타나면 → 결함 의심
```

결함이 있으면 에너지 일부가 결함에서 소비되어 바닥 에코 진폭도 낮아진다.

```
정상:      바닥에코 ████ (진폭 높음)
결함 있음:  결함에코 ██ + 바닥에코 ██ (바닥에코 진폭 감소)
```

실무에서는 재료 두께를 알고 있으므로, 검사 시작 전에 결함 게이트 + 두께 게이트를 모두 미리 설정해두고 스캔한다.

#### Gate A / Gate B — 결함 탐지·커플링 감시

```
신호강도
 │
 │  ██                 ██             ██
 │  ██   ┌── Gate A ───██───┐   ┌─Gate B─██─┐
 │  ██   │                  │   │           │
 │  ██   │                  │   │           │
 └──┴────┴──────────────────┴───┴───────────┴── 시간(깊이)
    ↑                 ↑                   ↑
 표면에코(IP)      결함에코           바닥면에코
  (Gate 밖)    (Gate A 초과→결함 감지)  (Gate B 감시)
```

| 게이트     | 역할                                                |
| ---------- | --------------------------------------------------- |
| **Gate A** | 결함 감시 — 표면 직후부터 바닥면 직전까지 설정      |
| **Gate B** | 바닥면 에코 감시 — 에코가 사라지면 커플링 이탈 경고 |

Gate B는 **탐촉자 접촉 상태(커플링)를 감시**하는 용도다. 바닥면 에코 위치에 Gate B를 설정해두면, 에코가 사라졌을 때 커플링 이탈로 판단할 수 있다.

> 두산에너빌리티: Gate B로 5채널 탐촉자 각각의 커플링 상태를 실시간 감시. 바닥면 에코 소멸 시 해당 채널 접촉 불량으로 판단 → 재스캔.

---

### 두께 측정용 게이트

두께를 측정할 때는 게이트를 두 개 사용한다.

```
진폭(%)
     │  ┌──┐              ┌───┐        ┌──┐
     │  │발신│              │   │        │  │
─────┼──┘  └──────────────┘   └────────┘  └──→ 시간
     │                         ↑               ↑
     │                      Gate A          Gate B
     │                    (T1 기록)        (T2 기록)
     │
     │  ├t┤←──── 2d/V ────→│
     │   ↑
     │  t_delay (커플링젤)
     │
     │  ←──────── T1 ──────→
     │  ←─────────────────── T2 ───────→
```

- **Gate A**: 1차 바닥 에코 구간 감지
- **Gate B**: 2차 바닥 에코 구간 감지 (초음파가 바닥면을 한 번 더 왕복한 에코)

#### 두께 계산식 — `(T2 - T1) ÷ 2 × 음속`

T1과 T2를 각각 분해하면:

```
T1  =  t_delay  +  2d/V   (커플링 지연 + 재료 1회 왕복)
T2  =  t_delay  +  4d/V   (커플링 지연 + 재료 2회 왕복)

T2 - T1  =  2d/V          ← t_delay 소거됨
```

`t_delay`(커플링 오차)가 자동으로 상쇄되므로, 커플런트 두께가 달라져도 두께 측정 결과에 영향을 주지 않는다.

```
d  =  (T2 - T1) × V ÷ 2
   =  (T2 - T1) ÷ 2 × 음속
```

### 결함 탐지 vs 두께 측정 비교

같은 A-scan에서 게이트를 어디에 놓느냐가 다르다.

```
결함 탐지:
[발신] ── [Gate A: 표면~바닥 사이] ───────────── [바닥에코]

두께 측정:
[발신] ──────────────────── [Gate A: 1차 바닥] ── [Gate B: 2차 바닥]
```

|                 | 결함 탐지        | 두께 측정            |
| --------------- | ---------------- | -------------------- |
| **게이트 위치** | 표면~바닥면 사이 | 바닥 에코 위치       |
| **게이트 개수** | 보통 1개         | 2개 (Gate A, Gate B) |
| **읽는 값**     | 진폭(%)          | TOF (T1, T2)         |
| **판단 기준**   | 임계값 초과 여부 | T2 - T1 시간 차      |

동시에 두 목적을 수행할 때는 게이트 3개를 쓴다 (결함용 1개 + 두께용 2개).

### 게이트 설정 공통 사항

#### 바닥 에코 위치를 어떻게 알고 게이트를 설정하는가

두 가지 방법이 있다.

**방법 1 — 음속과 두께로 위치를 계산**

재료 두께를 설계도면으로 미리 알고 있으면, 에코가 언제 돌아올지 계산할 수 있다.

```
바닥 에코 도착 시간  =  (두께 × 2) ÷ 음속

예) 탄소강 20mm 두께
  =  (0.020m × 2) ÷ 5,920 m/s
  =  약 6.76 μs
```

**방법 2 — A-scan 화면을 직접 보면서 설정 (더 일반적)**

탐촉자를 재료에 올려놓으면 A-scan에 에코가 바로 나타난다. 바닥 에코는 보통 **가장 크고 일정하게 나타나는 피크**이므로 쉽게 식별된다.

```
진폭│  ┌발신┐                    ┌───┐
   │  └───┘                    │   │  ← 바닥 에코
   └──────────────────────────────→ 시간
                                ↑
                          여기에 Gate A 설정
```

대부분의 UT 장비는 음속을 입력하면 x축이 **시간(μs) → 깊이(mm)** 로 자동 변환된다. 20mm 두께 재료면 20mm 위치에 에코가 표시되므로, 그 위치에 게이트를 놓으면 된다.

**실제 설정 순서**

```
① 장비에 재료 음속 입력 (탄소강 5,920 m/s 등)
        ↓
② 두께를 정확히 아는 교정 블록(Calibration Block)에 탐촉자를 댐
        ↓
③ A-scan에서 에코 위치 확인 → 예상 깊이와 맞는지 검증
        ↓
④ 에코가 나타나는 위치에 Gate A, Gate B 설정
        ↓
⑤ 실제 검사 대상으로 이동하여 스캔 시작
```

교정 블록을 사용하는 이유는, 실제 검사 대상에는 결함이 있을 수 있어 순수한 바닥 에코만 따로 확인하기 어렵기 때문이다.

#### 게이트 위치와 실제 에코 위치가 어긋나는 경우

에코 도착 시간은 `TOF = 거리 ÷ 음속`으로 결정되므로, 음속이 바뀌면 에코 위치가 이동한다.

**원인 1 — 온도 변화**

온도가 높아지면 재료 내 음속이 느려진다.

```
온도 낮을 때:   [Gate A]──[에코]──     (정상)
온도 올라갈 때: [Gate A]────────[에코]  (음속 느려짐 → 에코 오른쪽으로 밀림)
```

**원인 2 — 재료 두께 편차**

부식·마모·용접 등으로 두께가 균일하지 않으면 위치마다 에코 도착 시간이 달라진다.

**원인 3 — 커플링 불량**

에코가 게이트 안에 있어도 진폭이 임계값을 못 넘으면 감지되지 않는다.

| 상황               | 대처 방법                                                |
| ------------------ | -------------------------------------------------------- |
| 온도 편차가 클 때  | 음속 보정, 또는 게이트 폭을 넓게 설정                    |
| 두께 편차가 클 때  | 구간별 게이트 재설정, 에코 추적(Gate Tracking) 기능 사용 |
| 커플링 불안정할 때 | 임계값을 낮추거나 커플링 경보 기능 활용                  |

**에코 추적(Gate Tracking)**: 에코 위치를 자동으로 따라가도록 게이트가 이동하는 기능. 두께 편차가 큰 구간 검사 시 유용하다.

### 표면 에코와 다중 반사

**표면 에코는 어디 있나**

직접 접촉법(탐촉자를 재료에 바로 붙이는 방식)에서는 **표면 에코가 발신 펄스와 거의 동시에 발생**해서 A-scan 맨 앞에 합쳐져 보인다.

```
진폭│
   │  ┌─────┐                    ┌───┐
   │  │발신  │                    │   │  ← 바닥 에코
   │  │+표면 │                    │   │
   │  └─────┘                    └───┘──→ 시간
   │     ↑
   │  탐촉자가 재료에 붙어있어서 표면까지 거리 ≈ 0
   │  → 발신 펄스에 묻혀버림
```

단, **수침법(탐촉자와 재료 사이에 물이 있는 방식)** 에서는 물을 통과하는 시간이 생기므로 표면 에코가 별도로 보인다.

```
수침법 A-scan:

진폭│  ┌발신┐  ┌표면에코┐        ┌바닥에코┐
   │  └───┘  └───────┘        └────────┘──→ 시간
   │               ↑                ↑
   │          물/재료 경계면      재료 바닥면
```

**한 펄스가 재료 안에서 움직이는 과정**

```
탐촉자
──────  ← 경계면 A (탐촉자/재료)
재료
──────  ← 경계면 B (재료/바닥)
```

```
① 발사      →→→→→↓  재료로 들어감
② 바닥 도착         ↓ → 대부분 반사 ↑, 일부는 투과해서 사라짐
③ 표면 도착         ↑ → 탐촉자가 수신 (1차 바닥 에코), 일부는 다시 반사 ↓
④ 바닥 도착         ↓ → 또 반사 ↑, 또 일부 사라짐
⑤ 표면 도착         ↑ → 탐촉자가 수신 (2차 바닥 에코), 일부는 다시 반사 ↓
⑥ 이후 반복 → 에너지가 잡음 수준으로 사라질 때까지 계속
```

에너지가 줄어드는 이유는 세 가지다.

- **경계면 투과 손실**: 경계면을 지날 때마다 일부가 투과해서 빠져나감
- **재료 내 감쇠**: 재료를 지나는 동안 결정립 산란·흡수로 에너지 소모
- **빔 발산**: 탐촉자에서 멀어질수록 빔이 퍼져 단위 면적당 에너지 감소

초음파가 결함을 만나면 에너지가 **둘로 나뉜다.**

```
탐촉자
────────────
    ↓ 발사

    ●  결함
   ↑↓         ← 일부 반사 → 결함 에코 발생
    ↓          ← 일부 통과 → 계속 진행

────────────  바닥
    ↑          ← 통과된 에너지가 바닥에서 반사 → 바닥 에코 발생 (단 약해짐)
```

결함이 있어도 **바닥 에코는 사라지지 않는다.** 다만 결함이 에너지 일부를 흡수하므로 바닥 에코가 정상보다 약해진다.

```
결함 없을 때:  발신 ──────────── 바닥1(강함) ──── 바닥2 ──→
결함 있을 때:  발신 ── 결함1 ──── 바닥1(약함) ──── 결함2 ── 바닥2(더 약함) ──→
```

**바닥 에코가 갑자기 약해지는 것 자체가 결함의 신호다.** 결함 에코가 뚜렷하지 않더라도 바닥 에코 감소만으로 결함을 의심할 수 있다.

초음파는 바닥면과 표면 사이를 계속 왕복하므로 A-scan에는 2차, 3차 바닥 에코도 나타난다. 매번 반사될 때마다 에너지 일부가 손실되어 진폭이 점점 작아진다.

```
진폭│
   │  ┌──┐        ┌─┐       ┌──┐      ┌─┐    ┌┐
   │  │  │        │ │       │  │      │ │    ││  ← 점점 약해짐
   │  └──┘        └─┘       └──┘      └─┘    └┘──→ 시간
   │   ↑           ↑         ↑         ↑      ↑
   │ 발신+표면   결함에코   바닥1     바닥2  바닥3...
   │  에코       (깊이에    Gate A   Gate B
   │  (t=0)     따라 가변)   T1        T2
   │
   │  ※ 직접 접촉법: 발신+표면에코가 합쳐져 보임
   │  ※ 결함 있으면: Gate A(바닥1)부터 약해짐
```

#### 펄스 발사 방식과 타이밍

초음파는 연속으로 쏘는 것이 아니라 **짧은 펄스를 반복 발사**한다. A-scan 한 장에 보이는 1차·2차·3차 에코는 모두 **같은 하나의 펄스**가 왕복한 결과다.

다음 펄스는 이전 펄스의 에코가 사라진 뒤에 발사되므로 겹치지 않는다. 이 타이밍은 작업자가 설정하는 **검사 범위(Range)** 에 의해 자동으로 결정된다.

```
범위 = 100mm 설정 시:

[펄스 발사] → 에코 대기(33.8μs) → [다음 펄스 발사] → 반복
              ↑
     A-scan 창이 이 시간만큼 열림
     창이 닫히면 다음 펄스 발사
```

#### Range가 A-scan에 보이는 에코 개수를 결정한다

Range(검사 범위)는 A-scan 창의 너비다. 창이 어디까지 열리느냐에 따라 몇 번째 바닥 에코까지 보이는지가 달라진다.

```
두께 = 20mm 예시 (강재, 음속 5,920m/s → 바닥 에코 간격 ≈ 6.8μs)

 Range = 20mm (두께와 같음):
 진폭│  ┌─┐            ┌──┐
    │  │ │            │  │← 창 끝
    └──┘ └────────────┘  └──→
         발신+표면       바닥1
                        (창 끝에 딱 맞음 → 에코 1개만 보임)

 Range = 50mm (두께의 2.5배):
 진폭│  ┌─┐      ┌──┐    ┌─┐   ┌┐
    │  │ │      │  │    │ │   ││← 창 끝
    └──┘ └──────┘  └────┘ └───┘└→
         발신   바닥1   바닥2  바닥3
         +표면  (약해짐)(약해짐)
                        (창이 넓음 → 에코 2~3개 보임)
```

| Range 설정    | 보이는 바닥 에코 수 | 용도                           |
| ------------- | ------------------- | ------------------------------ |
| ≈ 두께        | 1개                 | 결함 탐지(Gate A 1개로 충분)   |
| ≈ 두께의 2배  | 2개                 | 두께 측정(Gate A·B 간격 확인)  |
| 두께의 3배+   | 3개 이상            | 다중 반사 패턴 확인, 감쇠 분석 |
| 두께보다 좁게 | 0개 (에코 창 밖)    | 유령 에코 위험 — 사용 금지     |

> 두께 측정 목적이라면 Range를 두께의 **약 2배**로 설정해 Gate A(바닥1)와 Gate B(바닥2)가 모두 창 안에 들어오도록 한다. Range = 두께로만 설정하면 바닥2가 창 밖으로 나가 Gate B를 잡을 수 없다.

## 매개체(매질)의 필요성

빛(전자기파)은 진공 속에서도 전파되지만, **초음파는 반드시 매질이 있어야 전파된다.**  
초음파는 매질 입자들이 서로 충돌하며 진동 에너지를 이웃 입자에게 전달하는 방식으로 나아가기 때문이다.

```
❌ 진공: 입자 없음 → 초음파 전파 불가

매질 있음 (고체·액체·기체):
  ○ → ○ → ○ → ○ → ○   (입자들이 진동을 이웃에게 전달)
  초음파 전파 가능
```

| 매질 종류 | 전파 가능 여부     | 비고                       |
| --------- | ------------------ | -------------------------- |
| 고체      | ✅ 종파·횡파 모두  | 결정립·탄성 구조로 잘 전파 |
| 액체      | ✅ 종파만          | 전단력 없어 횡파 불가      |
| 기체      | ✅ 종파만, 감쇠 큼 | UT 검사에 거의 사용 안 함  |
| 진공      | ❌                 | 입자 없어 전파 불가        |

> 이 때문에 탐촉자와 검사체 사이에 커플링젤(접촉매질)을 반드시 발라야 한다.  
> 탐촉자-검사체 사이 공기층이 생기면 초음파가 거의 100% 반사되어 검사체 안으로 들어가지 못한다.

## 매질 특성에 따른 전달 속도

초음파의 전달 속도는 매질의 **밀도**와 **탄성계수**에 따라 결정된다.

```
V = √(탄성계수 / 밀도)
```

- **탄성계수가 클수록 (딱딱할수록)** → 빠름
- **밀도가 클수록** → 느림

| 재질                | 종파 V_L  | 횡파 V_S  | 비고                   |
| ------------------- | --------- | --------- | ---------------------- |
| 탄소강              | 5,920 m/s | 3,230 m/s | 풍력 타워 주 재질      |
| 스테인리스강 (316L) | 5,740 m/s | 3,130 m/s | 두산에너빌리티 쉘 재질 |
| 인코넬 (625/718)    | 5,820 m/s | 3,050 m/s | 두산에너빌리티 쉘 재질 |
| 알루미늄            | 6,320 m/s | 3,130 m/s | —                      |
| 아크릴 (웨지)       | 2,730 m/s | 1,430 m/s | 경사각 탐촉자 웨지     |
| 물 (20°C)           | 1,480 m/s | —         | 커플링·수침법 기준     |
| 공기 (20°C)         | 343 m/s   | —         | 가장 느림              |

> 장비에 검사체의 음속을 정확히 입력해야 "초음파가 돌아온 시간 → 깊이" 변환이 맞는다. 음속이 틀리면 결함 깊이 표시가 틀린다.  
> **거리 계산**: `깊이 = 음속 × 돌아온 시간 / 2` (÷2는 왕복이므로)  
> **온도 보정**: 탄소강은 10°C 상승 시 종파 음속 약 −7 m/s 변화. 고온 검사 시 현장 실측 교정 필수.

## 전파 중 일어나는 현상

초음파가 매질을 통과하는 동안 다섯 가지 현상이 일어난다.

```
      초음파 빔
         ↓
  ───────────────────────── 경계면
         ↓  ↗  ↘  ↝  ↝↝
       투과 반사 굴절 흡수 산란
```

| 현상                    | 설명                                   | 검사에서의 의미                                      |
| ----------------------- | -------------------------------------- | ---------------------------------------------------- |
| **반사 (Reflection)**   | 두 매질 경계면에서 일부가 되돌아옴     | ★ 이 신호(에코)로 결함 탐지                          |
| **투과 (Transmission)** | 경계면을 통과해 진행                   | 반사율이 낮은 경계(재질 유사)에서 대부분 투과        |
| **굴절 (Refraction)**   | 두 매질의 음속 차이로 진행 방향이 꺾임 | 웨지·경사각 탐상의 원리                              |
| **흡수 (Absorption)**   | 매질 내 마찰로 에너지가 열로 변환      | 주파수↑, 두께↑ → 흡수↑ → 깊은 곳 탐지 어려움         |
| **산란 (Scattering)**   | 결정립·불순물에서 사방으로 흩어짐      | 조대 결정립 재질(오스테나이트계 SUS)에서 노이즈 증가 |

> **반사율은 두 매질의 음향 임피던스(Z) 차이**에 의해 결정된다.  
> 결함(균열·기공) 내부는 공기라 Z ≈ 0 → 강과 공기의 차이가 극단적으로 커 반사율 ≈ 100%.  
> 이것이 초음파 검사로 결함을 찾을 수 있는 핵심 원리.

---

## 파의 종류

### 종파 (Longitudinal Wave / L-wave)

입자가 **진행방향과 같은 방향으로** 진동한다. 스프링을 앞뒤로 밀면 조밀한 부분과 희박한 부분이 번갈아 퍼져나가는 것과 같다.

```
전파 방향 →

  조밀    희박    조밀    희박    조밀
  ●●●●   ○ ○   ●●●●   ○ ○   ●●●●
  ←→  입자 진동 방향 (전파 방향과 평행)
```

- 고체·액체·기체 모두 전파 가능
- **두산에너빌리티 Ch0** — 수직 종파로 수평 결함(라미네이션·내부 공동) 탐지

**종파 빔 경로 — 수직으로 내려갔다가 수직으로 돌아온다**

탐촉자를 재료 표면에 바로 붙이면 초음파가 수직으로 들어간다. 수평 결함(수평 균열·층상 결함)을 만나면 빔이 정면으로 부딪혀 그대로 되돌아온다.

```
탐촉자
└──┘
  ↓↑  빔(↓)과 에코(↑)가 같은 수직 경로
  ↓↑
──↓↑──────────────────  표면
  ↓↑
  ↓↑
  ━━━━━━  ← 수평 결함 (빔과 수직) → 에코 탐촉자로 돌아옴 ✅
```

**종파의 한계 — 비스듬한 결함은 에코가 옆으로 튀어나감**

결함이 완전히 수평이 아니라 약간 기울어져 있으면 에코가 탐촉자가 아닌 옆 방향으로 튀어나간다. 결함이 많이 기울수록 에코가 더 적게 돌아온다.

```
탐촉자
└──┘
  ↓
  ↓
──↓──────────────────  표면
  ↓
  ↓
   ╲━━━━  ← 비스듬한 결함
       ↘
        ↘→→→  에코가 옆으로 가버림 ❌ (탐촉자로 못 돌아옴)
```

이 경우 A-scan에서 결함 에코가 약하거나 아예 안 잡힌다. 대신 **바닥 에코가 약해지는 것**으로 결함을 의심할 수 있다.

---

### 횡파 (Shear Wave / S-wave)

입자가 **진행방향과 수직으로** 진동한다. 줄넘기 줄을 위아래로 흔들면 파동이 앞으로 나아가는 것과 같다.

```
전파 방향 →

  ↑       ↑       ↑
  ○   ○   ○   ○   ○   ○
      ↓       ↓       ↓
  ↕  입자 진동 방향 (전파 방향과 수직)
```

- **고체에서만** 전파 (액체·기체는 전단력이 없어 불가)
- 종파의 약 55% 속도 (탄소강 횡파: 3,230 m/s)
- **두산에너빌리티 Ch1~4** — 경사각 고정 횡파로 세로 방향 결함 탐지
- **풍력발전 PAUT** — S-scan으로 40°~70° 다각도 횡파 동시 발신

**왜 비스듬히 보내야 하는가**

종파(수직 빔)로는 **세로로 서있는 균열**을 찾지 못한다. 빔이 균열면과 평행하게 지나쳐버리기 때문이다.

```
수직 빔으로 세로 균열 탐지 시도:

탐촉자
└──┘
  ↓
──↓──────────────  표면
  ↓    ▎ ← 세로 균열
  ↓    ▎    빔이 균열 옆으로 그냥 지나침
  ↓    ▎    에코 없음 → 탐지 못함 ❌
──────────────────  바닥면
```

비스듬히 쏘면 균열 면에 정면으로 부딪혀 에코가 돌아온다.

**어떻게 비스듬히 보내는가 — 웨지(Wedge)**

탐촉자 자체는 수직으로만 소리를 낸다. 여기에 **아크릴로 만든 비스듬한 블록(웨지)** 을 붙이면 원하는 각도로 재료 안에 들어간다. 물에 꽂은 젓가락이 꺾여 보이는 것처럼, 아크릴→강 경계에서 소리가 굴절되면서 각도가 만들어진다.

```
탐촉자
└──────┘
│      │  ← 탐촉자 (수직으로 소리 발생)
 ╲    ╱
  ╲  ╱   ← 웨지 (아크릴 쐐기)
   ╲╱
────────  표면
    ╲
     ╲  ← 60° 각도로 재료 안에 들어감
```

**재료 안에서의 빔 경로 — 당구공처럼 튕김**

재료 안으로 들어간 빔은 당구공처럼 바닥면과 표면 사이를 대각선으로 튀어다닌다.

```
탐촉자+웨지
└──────┘
  ╲
   ╲              ╱  ← 반 스킵 지점
    ╲            ╱      (바닥 1회 반사 후 다시 올라옴)
─────╲──────────╱─────────────  표면
      ╲        ╱        ╲
       ╲      ╱          ╲
────────╲────╱────────────╲───  바닥면
         ╲  ╱              ╲
        1회 반사          2회 반사 (전 스킵)
  ├── 반 스킵 거리 ──┤
```

탐촉자를 앞뒤로 움직이면 빔이 닿는 위치가 바뀐다. 탐촉자를 적절한 위치에 놓아야 결함 에코가 돌아온다.

**에코가 돌아오는 경우 vs 돌아오지 않는 경우**

초음파가 결함면에 부딪혀 반사되는 것은 거울과 똑같다. 결함면이 빔과 수직이면 온 경로 그대로 되돌아오고, 기울어져 있으면 엉뚱한 방향으로 튀어나간다.

```
① 결함면이 빔과 수직 → 탐촉자로 돌아옴 ✅

탐촉자
└──┘
 ╲↑  빔(╲)과 에코(↑)가 같은 경로로 왕복
  ╲↑
───╲↑───────────  표면
    ╲↑
     ╲↑
      ▌ ← 결함 (빔과 수직)
        온 방향 그대로 되튕김
```

```
② 결함면이 기울어짐 → 에코가 엉뚱한 방향으로 가버림 ❌

탐촉자
└──┘
  ╲  빔
───╲────────────  표면
    ╲
     ╲
      ●  ← 빔이 기울어진 결함면에 부딪힘
     ╱ ↘
    ╱   ↘→→→  에코가 이 방향으로 가버림 (탐촉자 아님)
   ╱
  ╱  ← 기울어진 결함면
```

> 결함 방향이 빔과 맞지 않으면 에코가 안 잡힌다. 이 때문에 두산에너빌리티 Ch1~4처럼 여러 각도의 탐촉자를 써서 다양한 방향의 결함을 커버한다.

### 종파 vs 횡파 비교

| 구분         | 종파                        | 횡파                       |
| ------------ | --------------------------- | -------------------------- |
| 진동 방향    | 전파방향과 평행             | 전파방향과 수직            |
| 전파 매질    | 고체/액체/기체              | 고체만                     |
| 상대 속도    | 기준 (빠름)                 | 종파의 약 55%              |
| 빔 방향      | 수직 (표면과 90°)           | 경사각 (45°·60°·70° 등)    |
| 잘 잡는 결함 | 수평 결함 (라미네이션·층상) | 세로 방향 결함 (용접 균열) |
| 못 잡는 결함 | 세로 균열                   | 수평 결함                  |

## 커플링 — 탐촉자를 검사체에 어떻게 붙이나

탐촉자를 그냥 검사체에 갖다 대면 사이에 공기층이 생긴다. 공기는 음향 임피던스가 거의 0이라 초음파가 **99% 이상 반사**되어 검사체 안으로 거의 들어가지 못한다.

이를 해결하기 위해 **커플링젤(접촉매질)** 을 사이에 바른다.

```
❌ 젤 없이                    ✅ 젤 있음

탐촉자                        탐촉자
└───┘                         └───┘
 공기층 → 초음파 대부분 반사    젤층  → 초음파 통과
═══════════ 검사체             ═══════════ 검사체
  거의 안 들어감                  잘 들어감
```

| 커플링 매질                | 사용 상황                                          |
| -------------------------- | -------------------------------------------------- |
| 물 (로봇 탑재 물펌프 공급) | **풍력 타워 PAUT** — 로봇 이동 중 자동 공급 (확정) |
| 초음파 젤 (수용성)         | 일반 현장 접촉 검사 — 가장 많이 사용               |
| 물 (수침법)                | 자동화 검사·수조                                   |
| 글리세린                   | 수직면·오버헤드 검사                               |
| 엔진오일                   | 거친 표면, 고온                                    |

> 두산에너빌리티 쉘 검사: 탐촉자 5개가 스프링 텐션으로 쉘 표면에 밀착 → 젤이 균일하게 퍼져야 커플링 유지. Gate B(바닥면 에코)로 커플링 손실 실시간 감시.

## 음향 임피던스 — 왜 결함에서 에코가 강하게 돌아오나

음향 임피던스(Z)는 재질이 초음파를 얼마나 "버티는가"를 나타내는 값이다.  
**두 재질의 Z 차이가 클수록 경계면에서 반사율이 높아진다.**

```
Z = 밀도(ρ) × 음속(V)
```

| 재료          | 음향 임피던스 (×10⁶ Rayl) |
| ------------- | ------------------------- |
| 강            | 45.0                      |
| 알루미늄      | 17.0                      |
| 아크릴        | 3.2                       |
| 물 (커플링젤) | 1.48                      |
| 공기          | 0.0004                    |

| 경계                       | 반사율     | 의미                       |
| -------------------------- | ---------- | -------------------------- |
| 강 → 공기 (균열·기공 내부) | ≈ **100%** | 에코 매우 강함 → 결함 탐지 |
| 강 → 슬래그                | ≈ 8.2%     | 중간 에코                  |
| 강 → 커플링젤              | ≈ 87.6%    | 표면 에코 (초기 펄스)      |

> 균열·기공 안은 공기 = Z ≈ 0. 강과 공기의 Z 차이가 극단적으로 크기 때문에 초음파가 거의 100% 반사된다. 이것이 초음파 검사로 결함을 찾을 수 있는 핵심 원리.

---

## 경계면에서의 현상

초음파가 두 재질의 경계면에 부딪히면 세 가지 현상이 동시에 일어난다.

| 현상                       | 설명                                        |
| -------------------------- | ------------------------------------------- |
| 반사 (Reflection)          | 일부가 되돌아옴 → 이 신호로 결함 탐지       |
| 굴절 (Refraction)          | 일부가 통과하면서 방향이 꺾임               |
| 모드변환 (Mode Conversion) | 종파↔횡파 전환 — 웨지가 이 원리로 횡파 생성 |

```
          입사파 (종파)
             ↘ θ_i
재질 1  ───────────────────────
재질 2  ───────────────────────
             ↗ θ_r   ↘ θ_t
          반사파       굴절·투과파
                       (V₂ > V₁이면 θ_t > θ_i 로 꺾임)
                       ※ 모드변환: 종파 → 횡파 가능

결함(균열·기공):  Z₂ ≈ 0 (공기) → 반사율 ≈ 100% → 강한 에코
```

**모드변환이 실무에서 중요한 이유:** 경사각 탐촉자의 아크릴 웨지는 이 원리를 이용해 종파를 횡파로 바꾼다. 웨지 각도를 조절하면 검사체 안에서 원하는 각도의 횡파를 만들 수 있다.

---

## 경사각 검사가 왜 필요한가

수직 빔만으로는 용접부 결함을 찾지 못하는 경우가 많다. 이유는 결함의 방향 때문이다.

```
❌ 수직빔 — 수직 균열 탐지 실패

  탐촉자
  └──┘
   ↓ 수직 빔
─────│──────────  표면
     │
     │ ▎← 수직 균열 (빔과 평행)
     │ ▎
     │        → 빔이 균열면을 따라 지나쳐버림
─────────────────  바닥면              → 반사파 돌아오지 않음

✅ 경사각빔 — 수직 균열 탐지 성공

  탐촉자
  └──┘
   ↘ 경사각 빔 (예: 60°)
─────╲──────────  표면
      ╲
      ╲▎← 수직 균열 (빔이 정면으로 부딪힘)
       ↗
  반사파가 탐촉자로 돌아옴  → 에코 감지
```

| 검사 대상                   | 적합한 빔                |
| --------------------------- | ------------------------ |
| 라미네이션 (수평 층상 결함) | 수직 빔 (종파 0°)        |
| 용접 균열·LOF (세로 방향)   | 경사각 빔 (횡파 45°~70°) |
| 루트부 미용접 (바닥 근처)   | 경사각 Full Skip         |

> 두산에너빌리티 5채널 구성이 바로 이 이유다: Ch0(수직 종파)로 수평 결함을, Ch1~4(경사각 횡파 상하좌우)로 모든 방향의 수직 면형 결함을 커버한다.

## A-scan 읽는 법

탐상기 화면의 기본 파형. X축은 시간(=깊이), Y축은 신호 강도(에코 높이).

```
탐상기 ─(전기 펄스)→ 압전소자 ─(초음파)→ 검사체
탐상기 ←(전기 신호)─ 압전소자 ←(반사파)─
```

```
신호강도(%)
        │
  100%  │  ██                        ██
        │  ██          ██            ██
   20%  │  ██          ██            ██
        └──┴───────────┴─────────────┴── 시간(깊이)
           ↑            ↑             ↑
        초기펄스       결함에코      바닥면에코
       (탐촉자 직하)  (결함 깊이)   (재료 두께)
```

- **초기펄스**: 탐촉자에서 초음파가 출발하는 순간 (항상 맨 왼쪽)
- **바닥면 에코**: 재료 바닥에서 반사 (항상 맨 오른쪽, 재료 두께 위치)
- **결함 에코**: 결함이 있으면 바닥면 에코 **앞(왼쪽)** 에 추가 에코 출현
- 결함 에코의 **X축 위치** → 결함 깊이, **Y축 높이** → 결함 크기 추정

## 게인 (Gain) — 신호 증폭

### 증폭이 필요한 이유

탐촉자 내부의 압전소자가 에코(음압)를 받으면 **전기 신호(전압)** 로 변환한다. 그런데 이 전압이 **µV(마이크로볼트) 수준으로 매우 작다.**

```
에코 도달
    │
    ▼
압전소자 → 전기 신호 생성 (0.001 ~ 수 mV 수준)
                │
                ↓ 이 상태론
                - 화면에 표시 불가 (너무 작음)
                - ADC가 정확하게 변환 불가
                - 노이즈에 묻혀버림
```

마이크가 소리를 잡으면 아주 약한 전기 신호가 나오듯, 이 신호를 **읽을 수 있는 크기로 키우는 작업이 증폭(Amplification)** 이다.  
게인(Gain)은 이 증폭의 크기를 조절하는 볼륨 노브에 해당한다.

---

### dB (데시벨) — 증폭 단위

몇 배 키웠는지를 곱셈 대신 **덧셈으로 표현**할 수 있게 만든 단위다.

```
+6 dB  → 진폭 2배
+20 dB → 진폭 10배
+40 dB → 진폭 100배

증폭 30dB + 증폭 20dB = 증폭 50dB (그냥 더하면 됨)
```

현장에서 "게인을 6dB 올렸다" = "신호를 2배 키웠다" 는 뜻이다.

---

### 증폭 3가지 — 신호 경로 위치별

신호가 탐촉자에서 화면까지 오는 경로에서 증폭이 세 단계에 걸쳐 일어난다.

```
탐촉자(에코 수신)
    │ 아날로그 전압 신호 (µV~mV)
    ▼
[아날로그 증폭] ← 하드웨어 회로(op-amp)
    │ 증폭된 아날로그 신호
    ▼
[ADC] — 아날로그 → 디지털 변환
    │ 디지털 값 (숫자)
    ▼
[디지털 증폭] ← FPGA/DSP에서 곱셈
    │
    ▼
[소프트웨어 증폭] ← CPU/펌웨어에서 처리
    │
    ▼
A-scan 화면
```

|           | 아날로그 증폭         | 디지털 증폭      | 소프트웨어 증폭  |
| --------- | --------------------- | ---------------- | ---------------- |
| 적용 위치 | ADC 이전              | ADC 이후 (FPGA)  | ADC 이후 (CPU)   |
| 방식      | 전기 회로로 전압 증폭 | 디지털 값 × 계수 | 디지털 값 × 계수 |

**아날로그 증폭이 가장 중요한 이유 — ADC 전에 신호를 충분히 키워야 한다:**

```
아날로그 증폭 충분:          아날로그 증폭 부족:
    │  ████                      │  █
    │  ████  → ADC               │  █  → ADC → 디지털 증폭
    │  ████                      │  █              │  ██
                                                   │  ██  ← 계단 노이즈
                                                   │       (해상도 손실)
ADC가 신호를 세밀하게 잡음     ADC가 뭉개서 변환
                               → 이후 디지털/소프트웨어 증폭으로 복구 불가
```

ADC 이전 단계에서 신호가 충분히 증폭되지 않으면, 이후 어떤 방법으로도 잃어버린 정밀도를 되살릴 수 없다.

> erut 시스템에서 `gainsSw`는 소프트웨어 증폭, `gainsDigital`은 디지털 증폭에 해당한다.  
> 둘 다 ADC 이후 단계이므로 기본 신호 품질은 하드웨어(아날로그) 단계에서 결정된다.

---

### 게인 설정 기준

```
게인 낮음 (예: 30 dB)       게인 높음 (예: 50 dB)

신호│                        신호│
   │  █                         │  ████
   │  █   █                     │  ████  ████
   └──────────→ 시간             └──────────────→ 시간
   작은 결함 에코 안 보임        작은 결함 에코도 보임
                                 (단, 노이즈도 같이 커짐)
```

- **기준 게인**: 교정 블록(SDH)에서 에코를 80% 높이로 맞춘 값
- **검색 게인**: 기준 게인 + 6 dB (더 민감하게 설정, 스캔 시 사용)
- 지시 발견 → **기준 게인으로 돌아와서** 에코 높이 평가

> 게인을 너무 높이면 노이즈(거짓 신호)가 결함처럼 보일 수 있다.

## 근거리 음장 (Near Field)

탐촉자 바로 앞 구간은 여러 소자에서 나온 초음파가 서로 간섭해 음압이 불규칙하게 출렁인다. 이 구간에서는 결함 신호가 실제보다 크게도, 작게도 보일 수 있어 **크기 정량 평가를 신뢰할 수 없다.**

```
N = D² / 4λ    (D: 진동자 직경, λ: 파장)
```

```
┌──────┐
│ 탐촉자│
└──┬───┘
   ↓↓↓
   │←── 근거리 음장 N ──┤←────── 원거리 음장 ──────→
   │                    │
음압│  ╱╲╱╲╱╲╱╲╱╲       │╲
   │                    │  ╲_______________
   │  불규칙 (간섭)      N   단조 감소 (신뢰 구간)
   │  정량 평가 ✕               정량 평가 ○
```

> 교정에 사용하는 SDH(가로 구멍)가 반드시 근거리 음장 **밖**에 있어야 교정값이 신뢰할 수 있다.

## 감쇠 (Attenuation)

초음파는 재료를 통과할수록 에너지가 줄어든다. 깊은 결함일수록, 두꺼운 재료일수록 에코가 약해진다.

**원인:**

- **흡수**: 매질 내 마찰로 에너지가 열로 변환 — 주파수가 높을수록 커짐
- **산란**: 결정립·불순물에서 초음파가 사방으로 흩어짐

```
탐촉자                              전파 거리 →
 │  I₀ (100%)    I₁ (↓)    I₂ (↓↓)
 │  ██████████   ███████   ████
 │                ↗↗ ↗↗ ↗            ← 산란: 사방으로 분산
 │  ─────────────────────────────→

 ∴ 두꺼운 재료·고주파일수록 감쇠 커짐 → 저주파 탐촉자 사용
```

**재질별 감쇠 계수 (5MHz 기준):**

| 재질                      | 감쇠 계수     | 비고                   |
| ------------------------- | ------------- | ---------------------- |
| 탄소강                    | 0.5 ~ 2 dB/cm | —                      |
| 오스테나이트 스테인리스강 | 2 ~ 10 dB/cm  | 조대 결정립, 산란 많음 |
| 주철                      | 5 ~ 20 dB/cm  | —                      |

깊이에 따른 감쇠를 자동 보정하는 것이 **TCG (Time Corrected Gain)** 다. 얕은 곳과 깊은 곳의 동일한 결함 크기의 SDH 에코 높이를 동일하게 보정한다.

```
TCG 없음:              TCG 있음:
신호│                  신호│
   │ ████              │ ████  ████  ████
   │ ████  ███         │              (깊이 달라도
   │ ████  ███  ██     │               에코 높이 동일)
   └──────────────→    └──────────────→
     얕은곳 깊은곳       깊이 무관 균등
```

> 두산에너빌리티 SUS/인코넬 쉘: 탄소강보다 감쇠가 2~5배 크다. 교정 시 TCG 설정 필수, 주파수도 낮게 선택.
